## NODE - SERVER

### Iniciar Proyecto
#### Clonar Repositorio
~~~
git clone https://github.com/evilsmoon/node-server.git
cd node-clima
npm i
~~~
### NPM INSTALL
~~~
"dependencies": {
  "express": "^4.17.1"
},
"devDependencies": {
  "nodemon": "^2.0.9"
},
~~~

### Start Server

~~~
node server.js
~~~
### Start DevServer

~~~
nodemon server.js
~~~
